# npm 명령어

### npm init

package.json을 만드는 명령어

### npm install

패키지를 성치하는 명령어

npm install 패키지@버전 으로 특정 버전을 설치할 수 있다

npm install 주소 로 특정한 저장소에 있는 패키지를 설치할 수 있다.

-D or --save-dev 옵션으로 devDependencies에 추가

-g 옵션으로 글로벌 패키지 설치

### npm update

패키지를 업데이트하는 명령어

### npm dedupe

npm에서 중복된 패키지를 정리할 때 사용

### npm docs

패키지에 대한 설명을 보여줌
